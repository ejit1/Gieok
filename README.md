# Gieok_여행 명소 제공 서비스

<br>

## 1. 주요 기능
- 등록된 여행 명소, 베스트 사진, 명소 순위 통함 검색
- 특정 지역 선택 시 해당 지역의 관광 명소 정보 제공
- 공지, 포토, 이벤트 등록 및 확인
- 여행을 위한 동행 신청 및 수락 기능 제공

<br>

## 2. 개발 환경 및 배포 URL

**2-1. 개발 환경**
- Language
Java, JSP, HTML, CSS, JavaScript, jQuery
- Framework
Spring Framework (Legacy), Mybatis
- DBMS
Oracle
- API
Summernote API, Kakao Map / 주소 API
- OS
Linux

**2-2. 배포 URL**
gieok.icu
www.gieok.icu
